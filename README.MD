# How to get it working locally
### Install Node
[Download](https://nodejs.org/en/download/) and install node.

### Clone this repo
``` git clone git@github.com:sebwong/lori-commercial.git ```

### Navigate to the dir
``` cd lori-commertial ```

### Install needed packages
``` npm install ```

### Start
``` npm start ```

### Browse

Go to ``` localhost:3000 ``` in your browser
